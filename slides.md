# Airbnb ì‚¬ë¡€: Hypernova

> A service for server-side rendering your JavaScript views

Server

```js {all|5-10}
var hypernova = require('hypernova/server');

hypernova({
  devMode: true,
  getComponent(name) {
    if (name === 'MyComponent.js') {
      return require('./app/assets/javascripts/MyComponent.js');
    }
    return null;
  },
  port: 3030,
});
```
